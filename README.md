# 🚀 Raw Node.js Learning Journey

A personal learning repository where I build a Node.js server from scratch using **TypeScript** and the native **HTTP module**, without using Express.js.

The main goal of this project is to understand how Node.js works behind the scenes before relying on frameworks such as Express.js, NestJS, or Fastify.

---

## 🎯 Goals

* Understand the Node.js runtime
* Learn how HTTP servers work
* Handle requests and responses manually
* Build a routing system from scratch
* Parse request bodies
* Create CRUD APIs
* Work with the File System
* Improve TypeScript skills
* Learn backend fundamentals deeply

---

## 🛠 Technologies Used

* Node.js
* TypeScript
* TSX
* Native HTTP Module

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/raw-nodejs-learning.git
```

Navigate into the project:

```bash
cd raw-nodejs-learning
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

---

## 🧠 Why This Repository?

I previously worked with the MERN stack and built applications using Express.js.

This repository is my effort to learn how Node.js works internally by building everything manually with the native HTTP module and TypeScript.

The goal is not just to build APIs, but to understand what frameworks are doing behind the scenes.
